#### 1. 산술 연산자

- ##### + , - , * , / 

- ##### = : 대입(할당) 연산자

- ##### % : 나머지 값을 구하는 연산자 → 배수, 짝홀수..... 

- ##### 자리수 구하기

  - 123 % 10 -> 3 

  - 123 / 10 → 12 

  - 12 % 10 → 2 

  - 12 / 10 → 1

```c
int main() {

int v1 = 10;
int v2 = 3;
printf("v1 : %d, v2 : %d\n", v1, v2);
printf("/n");

printf("%d + %d = %d\n", v1, v2, v1 + v2);
printf("%d - %d = %d\n", v1, v2, v1 - v2);
printf("%d * %d = %d\n", v1, v2, v1 * v2);
printf("%d / %d = %d\n", v1, v2, v1 / v2);
printf("%d %% %d = %d\n", v1, v2, v1 % v2);
}
```



#### 2. 복합 대입 연산자

- ##### += , -=

  a += b -> a = a + b : a 와 b 를 덧셈한 후에 나오는 결과를 왼쪽에 대입

```c
int main() {

	int v3, v4;
	v3 = v4 = 5;
	printf("v3 : %d, v4 : %d\n", v3, v4);
	printf("\n");

	printf("v3 += 1 : %d\n", v3 += 1);
	printf("v3 -= 1 : %d\n", v3 -= 1);
	//5 += 1; Error
	// - 계산후에 나온 결과값을 왼쪽에 넘겨야 하기때문에 왼쪽에는 변수가 위치해야 합니다.

	printf("v3 *= v4 : %d\n", v3 *= v4);

}
```



#### 3. 관계(비교) 연산자

- ##### > , < , >= , <=

  - == : 같을때가 참
  - != : 왼쪽과 오른쪽이 서로다를때가 참

- ##### 데이터의 대소 관계를 비교하는 연산자
 - C 언어에서는 참과 거짓을 1과 0으로 나타냄 ( 참 : 1 , 거짓 : 0 )
- C 언어에서는 0 이외의 모든값은 참

```c
int main() {

	double v5 = 3.1;
	double v6 = 3.5;
	printf("v5 : %.1f, v6 : %.1f\n", v5, v6);
	printf("\n");

	printf("v5 <= v6 : %d\n", v5 <= v6);
	printf("v5 == v6 : %d\n", v5 == v6);
	printf("v5 != v : %d\n", v5 != v6);

}
```



#### 4. 논리연산자

- ##### && ( and 연산 )

  - 조건식 _ a && 조건식 _ b
  - 조건식 _ a , 조건식 _ b 둘다 참일때만 참

- ##### || ( or 연산 )

  - 조건식 _ a || 조건식 _ b
  - 조건식 _ a, 조건식 _ b 둘중에 하나라도 참이면 참

- ##### ! ( not 연산 )

  - !( 조건식 )

  - 조건식의 결과가 참이면 거짓으로, 거짓이면 참으로 변경

```c
int main() {

	int v7 = 10;
	int v8 = 8;
	printf("v7 = %d, v8 = %d\n", v7, v8);
	printf("\n");

	printf("v7 == 10 && v8 > 10 : %d\n", v7 == 10 && v8 > 10); // 0 (false)
	printf("v7 == 10 || v8 > 10 : %d\n", v7 == 10 || v8 > 10); // 1 (true)
	printf(" v7 : %d\n", !v7); // 0 (false)

}
```



#### 5. 증감연산자

- ##### 값을 1 증가 시키거나, 1 감소 시킬때 사용

  - ++ : 1 증가
  - -- : 1 감소
  - ++n : 1 증가후에 다른 연산 
  - n++ : 다른 연산후에 1 증가 
  - --n : 1 감소후에 다른 연산 
  - n-- : 다른 연산후에 1 감소


```c
int main() {

	int v9 = 5;
	printf("v9 : %d\n", v9);
	printf("\n");

	int res;
	res = ++v9;
	printf("v9 : %d, res : %d\n", v9, res);

	res = v9++;
	printf("v9 : %d, res : %d\n", v9, res);

	res = --v9;
	printf("v9 : %d, res : %d\n", v9, res);

	res = v9--;
	printf("v9 : %d, res : %d\n", v9, res);

	v9 = 5;
	v9++;
	printf("v9 : %d\n", v9);

}
```
