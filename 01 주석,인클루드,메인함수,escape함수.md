### 1. 주석

| //   | 주석        | 프로그램 코드 안에 설명을 달아 줄 때 사용. |
| ---- | ----------- | ------------------------------------------ |
| //   | 슬래쉬 두번 | 한줄 주석                                  |
| /**/ | 슬래쉬 별   | 여러줄 주석                                |

### 2. 단축키

|    단축키    |   내용    |
| :----------: | :-------: |
| ctrl + k + c | 주석 처리 |
| ctrl + k + u | 주석 해제 |
|   ctrl + u   | ctrl + z  |
|   ctrl + y   | ctrl + y  |

### 3. 코드

```c
#include <stdio.h>
// 전처리기 : c언어에서 제공하는 명령어가 정의된 파일을 프로그램안에 포함 시킴

int main() // main 함수( 프로그램 시작 )
{ // main 함수 시작

printf("Hello World!!");
// 하나의 코드가 끝나는 마지막에는 ';' 기호를 작성

// 컴파일 ( compile ) : ctrl + F7
// - 프로그래밍 언어로 작성한 내용을 기계어로 변환하는 과정
// - 컴파일된 내용을 오브젝트 파일(.obj)에 저장함

// 링커 ( linker )
// - 오브젝트 파일과 헤더파일을 합쳐서 실행파일(.exe)을 생성

// 빌드 ( build )     : ctrl + shift + b
// - 컴파일과 linker 한번에 처리함

// 실행 : ctrl + F5

} // main 함수 종료
```
